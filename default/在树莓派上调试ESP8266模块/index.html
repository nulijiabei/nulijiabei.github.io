在树莓派上调试ESP8266模块

WiFi模块 ESP8266串口转WiFi/无线透传/工业级/安信可/ESP-01S 

设置树莓派使用PL011 UART串口通信

/boot/cmdline.txt  
    移除 console=serial0,115200 
/boot/config.txt 
    添加 enable_uart=1
    添加 dtoverlay=pi3-miniuart-bt

ESP8266 连接树莓派 UART 串口   

UART 1  3.3v -> ESP8266 3v3
UART 8  TxD  -> ESP8266 RX
UART 10 RXD  -> ESP8266 TX 
UART 6  GND  -> ESP8266 GND

安装串口工具
sudo apt-get install minicom

端口设置    
Serial port setup

| A -    Serial Device      : /dev/ttyAMA0                              |
| B - Lockfile Location     : /var/lock                                 |
| C -   Callin Program      :                                           |
| D -  Callout Program      :                                           |
| E -    Bps/Par/Bits       : 115200 8N1                                |
| F - Hardware Flow Control : No                                        |
| G - Software Flow Control : No 

注意: AT命令的结束不能用回车键，要用组合键Ctrl+M和Ctrl+J
